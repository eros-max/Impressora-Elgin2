# Impressora-Elgin2 - Guia de Manuseio
## 1° Etapa - Instalação
1. Instale e faça a extração dos arquivos necessários para a utilização do programa: CANC_SAT.xml, E1_Impressora01.dll, Java-Aluno EM.iml, Main.java, NFCe.xml, NFCeCancelamento.xml, README.md e XMLSAT.xml.
2. Instale o ambiente de desenvolvimento integrado para utilizar a linguaguem java, recomendação: IntelliJ IDEA (foi o programa utilizado).
3. Coloque-os numa pasta, se não já estiverem em uma, e abra no IntelliJ IDEA.
4. <img width="962" height="750" alt="Image" src="https://github.com/user-attachments/assets/26b73b39-3cf8-46ac-9719-60598f8a48a0" />
5. Selecione a pasta que possuir os arquivos desse modo:

## 2° Etapa - Substituições
1. Clique duas vezes sobre o "Main". Abra a pasta onde residem os arquivos, e abra as propriedades do arquivo "E1_Impressorra01.dll" usando o botão direito do mouse. Assim feito, copie o caminho do arquivo, e cole no respectivo local, mantendo o "//E1_Impressorra01.dll".
2. <img width="1737" height="797" alt="Image" src="https://github.com/user-attachments/assets/a36c5e79-f00a-49e0-a997-caf98d1bc62f" />

4. <img width="1918" height="1020" alt="Image" src="https://github.com/user-attachments/assets/889e7f17-b222-43c3-a0bc-bd03857f8d47" />

## 3° Etapa - Utilização
1. Após essas ações, volte ao início do código na linha 1 e leia os comentários deixados pelos programadores.
2. Após a leitura apropriada, aperte o botão para iniciar o programa:
3. <img width="1918" height="1017" alt="Image" src="https://github.com/user-attachments/assets/c74ae49f-b475-47dc-a708-aae8ea470f97" />
4. Logo após, o menu aparecerá na parte de baixo do programa, nessa parte também aparecem os erros que podem vir a acontecer:
5. <img width="1918" height="1016" alt="Image" src="https://github.com/user-attachments/assets/b142cbf5-3f8c-4235-8979-c08f39f9b148" />
6. Ao aparecer este menu, digite 1 para configurar a conexão e digite as demais requisições que o programa pedir, de acordo com sua máquina.
7. Feito isso, Digite 2 ao voltar do menu para abrir a conexão, se não for feito isso, não será possível usar as outras funções e a impressora.
8. **Faça uso de sua impressora!**
